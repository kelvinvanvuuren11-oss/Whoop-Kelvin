Connect your WHOOP.

This standalone reads your own WHOOP data. You'll need a developer app + a backend proxy (Netlify Functions, Vercel, etc.) because the WHOOP token endpoint doesn't allow direct browser calls.
  1. Go to developer.whoop.com → create an app
  2. Set redirect URL to YOUR-SITE-URL/ (where you'll host this file)
  3. Copy the Client ID and Client Secret
  4. Deploy a backend function that exchanges the OAuth code for tokens — use the netlify/functions/whoop-token.js example from this project (or any equivalent serverless function), set WHOOP_CLIENT_ID, WHOOP_CLIENT_SECRET, WHOOP_REDIRECT_URI as env vars
  5. Paste your Client ID + the proxy URL below
